# Random Quote Generator 📖

A web app that displays random inspirational quotes.

## Features
- 200+ quotes in 4 categories
- Dark and Light mode
- Save favorite quotes
- Share and Copy quotes
- Works on mobile too

## Technologies Used
- HTML
- CSS
- JavaScript
